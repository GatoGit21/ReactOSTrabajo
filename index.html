Introducción

ReactOS es un sistema operativo de código abierto cuyo código se rige por la licencia GPL[1]. Se basa en Windows Server 2003 y en su kernel de Windows NT 5.2[2], lo que le permite ejecutar aplicaciones creadas para Windows (desde Windows 95 a Windows 8[3]) aunque se encuentra en un estado de desarrollo constante.

Requisitos[4]:

  • Ram: Mínimo 64 MB, recomendado 256 MB.
  • CPU: x86 o x86_64, desde Pentium en adelante.
  • Mínimo 450 MB de espacio libre en Discos IDE o SATA. Los formatos soportados son FAT16 y FAT32.
  • GPU con puerto VGA.

Características:

Compatibilidad general con programas de 32 bits de Windows XP y Windows Server 2003, aunque se ha logrado ejecutar aplicaciones no compatibles con estas versiones como Minecraft[4], RPCS3[5] o VirtualBox[7]. En su versión de lanzamiento de 32 bits, es compatible con programas del día a día como LibreOffice, Firefox, VLC, LibreCAD [8] o Dev-C++ [9], aunque por ahora, algunos solo con versiones más antiguas.

Posee compatibilidad con controladores de hardware para Windows Server 2003 y XP para poder funcionar en hardware antiguo.

Su kernel es compatible con el lenguaje Bash, y puede ejecutar los .bat como si fuera Windows. Posee CMD pero actualmente no hay planes para replicar PowerShell.
